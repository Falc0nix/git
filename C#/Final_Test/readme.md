# Ниже приведен алгоритм заданной задачи с пояснениями:

## I. Ввод размера массива

    1. Задается чекпоинт для проверки валидности введенного значения размера массива
    2. Через консоль задается размер массива, который должен быть натуральным (то есть положительным и целым числом)
    3. Далее следует проверка задано ли в консоли ввода именно число (а не произвольный текст) и не пропустил ли пользователь ввод размера массива
    4. Если введено не число или пользователь ничего не ввел, программа "откатывается" к заданному чекпоинту и просит повторно ввести размер массива
    5. Далее следует проверка задано ли в консоли ввода именно положитель число (а не отрицательное и не 0)
    6. Если введено отрицательно число или 0, программа "откатывается" к заданному чекпоинту и просит повторно ввести размер массива

## II. Ввод элементов строкового массива размера, определенного в разделе I данной Справки

    1. Задается "пустой" строковый массив размера, определенного в разделе I данной Справки
    2. Через консоль поочередно задаются элементы строкового массива. Номер задаваемого элемента также выводится в запросе на ввод в консоли
    3. После ввода каждого элемента следует проверка, ввел ли пользователь элемент массива, или там пустое текстовое значение. Если пользователь ввел непустой элемент - переходим к п.4 раздела II, если не ввел - откатываемся к п.2 раздела II
    4. Если введено не число или пользователь ничего не ввел, программа "откатывается" к заданному чекпоинту и просит повторно ввести размер массива
    5. Далее следует проверка задано ли в консоли ввода именно положитель число (а не отрицательное и не 0)
    6. Если введено отрицательно число или 0, программа "откатывается" к заданному чекпоинту и просит повторно ввести размер массива

## III. Вывод посредством цикла всех элементов заданного массива на экран

## IV. Обработка заданного массива согласног условиям задачи с последующим выводом конечного массива на экрана экран

    1. Задается новый "пустой" строковый массив размера 0
    2. Следует поэлементный перебор исходного заданного масива согласно цикла.
    3. Если длина строкового элемента исходного массива меньше, либо равна 3, то размер конечного массива, сначала, увеличивается на 1, а потом - в новый элемент (на который массив увеличился), записывается значение исходнного массива, соответсвующего заданному условию
    4. Если длина строкового элемента исходного массива больше, программа по циклу переходит к следующему элементу исходного массива, и проверка из п.3 и п.4 раздела 4 повторяется, уже для нового элемента
    5. Конечный массив, длина всех элементов которого меньше, либо равна 3, выводится на экран посредством цикла
